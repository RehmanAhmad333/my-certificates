## 🧩 Tasks Completed (All 11)

### 📌 Core Tasks (Phase 1)

#### Task 1: Exploring and Visualizing a Simple Dataset
- **Dataset:** Iris Dataset
- **What I did:** Loaded data with pandas, performed EDA, created scatter plots, histograms, and box plots.
- **Skills:** Pandas, seaborn, matplotlib, outlier detection.

#### Task 2: Predict Future Stock Prices (Short-Term)
- **Dataset:** Yahoo Finance (Apple stock)
- **What I did:** Used yfinance to fetch historical data, trained Linear Regression & Random Forest to predict next day’s close.
- **Skills:** Time series, regression, API data fetching, actual vs predicted plots.

#### Task 3: Heart Disease Prediction
- **Dataset:** UCI Heart Disease (Kaggle)
- **What I did:** Data cleaning, EDA, Logistic Regression & Decision Tree classifiers, ROC curve, confusion matrix, feature importance.
- **Skills:** Binary classification, medical data analysis, ROC‑AUC.

#### Task 4: General Health Query Chatbot (Prompt Engineering)
- **Tools:** OpenAI GPT‑3.5 API
- **What I did:** Built a Python chatbot with prompt engineering (friendly medical assistant). Added safety filters to avoid harmful advice.
- **Skills:** Prompt design, LLM API integration, safety handling.

#### Task 5: Mental Health Support Chatbot (Fine‑Tuned)
- **Base Model:** DistilGPT2
- **Dataset:** EmpatheticDialogues (Facebook AI)
- **What I did:** Fine‑tuned using Hugging Face Trainer API to produce empathetic responses. Built Streamlit interface.
- **Skills:** Fine‑tuning, emotional tone design, conversation modeling.

#### Task 6: House Price Prediction
- **Dataset:** Kaggle House Prices
- **What I did:** Preprocessing (sqft, bedrooms, location), trained Linear Regression & Gradient Boosting, evaluated with MAE & RMSE.
- **Skills:** Regression, feature scaling, model evaluation.

### 🚀 Advanced Tasks (Phase 2)

#### Task 1: News Topic Classifier Using BERT
- **Dataset:** AG News (Hugging Face)
- **What I did:** Fine‑tuned `bert-base-uncased` for news headline classification. Deployed with Streamlit.
- **Metrics:** Accuracy & F1‑score.
- **Skills:** Transformers, transfer learning, NLP deployment.

#### Task 2: End-to-End ML Pipeline with Scikit-learn Pipeline API
- **Dataset:** Telco Churn
- **What I did:** Built full pipeline (scaling, encoding) + Logistic Regression & Random Forest. Used GridSearchCV. Exported with joblib.
- **Skills:** Production‑ready pipelines, hyperparameter tuning, model serialization.

#### Task 3: Multimodal ML – Housing Price Prediction Using Images + Tabular Data
- **Data:** Housing Sales + house images (public sources)
- **What I did:** Extracted image features using CNN (ResNet), fused with tabular data, trained regression model. Evaluated with MAE & RMSE.
- **Skills:** Multimodal learning, CNNs, feature fusion.

#### Task 4: Context-Aware Chatbot Using LangChain / RAG
- **Knowledge Base:** Custom corpus (Wikipedia pages)
- **What I did:** Built RAG chatbot with LangChain, vector store (FAISS), memory for conversation history. Deployed with Streamlit.
- **Skills:** Conversational AI, document embedding, retrieval‑augmented generation.

#### Task 5: Auto Tagging Support Tickets Using LLM
- **Dataset:** Free‑text support tickets
- **What I did:** Used zero‑shot, few‑shot, and fine‑tuned LLM to tag tickets into categories. Output top‑3 tags per ticket.
- **Skills:** Prompt engineering, LLM classification, few‑shot learning.
