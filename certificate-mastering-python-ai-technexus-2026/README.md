This 4-month professional training program covered a **complete AI development stack** – from Python basics to advanced AI agents. Below is the detailed curriculum I mastered.

- What is AI? (History, types, applications)
- Why learn AI in 2025? (Industry impact)
- AI Engineer Roadmap 2025
- AI vs ML vs Deep Learning (clear distinctions)
- AI Software Development Life Cycle (SDLC for AI systems)

### Module 2: FastAPI Backend for AI Applications
- FastAPI Hello World & Poetry setup
- Virtual environments for AI projects
- FastAPI routing deep dive
- Path, Query, and Body parameters
- Group routers & middleware
- File handling & exception management
- API Key authentication

- JWT authentication (Login/Signup)

### Module 3: Databases for AI
- SQL vs NoSQL (comparison, use cases)
- FastAPI + MongoDB with PyMongo

- Table Plus & PostgreSQL management

### Module 4: Data Preprocessing (Core AI Skill)
- NumPy – numerical computing, arrays, operations
- Pandas – data manipulation, cleaning, analysis
- OpenCV – image & video processing

### Module 5: Neural Networks & Deep Learning
- Neural Networks explained in-depth (architecture, backpropagation)
- Keras CNN – Cat vs Dog image classifier (complete project)

### Module 6: LLMs & Model Deployment
- What are LLMs? (GPT, Gemini, Claude)
- Pricing, settings, local deployment
- Prompt engineering fundamentals
- LLMs + FastAPI – build & render AI models

### Module 7: AI Agents (Advanced)
- What are AI Agents? (concepts, use cases)
- AI Agent development with OpenAI SDK (step-by-step)
- Deep dive into tool calling
- OpenAI Agent SDK + MongoDB integration
- OpenAI Agent SDK – local deployment & LLM context
- Design patterns for OpenAI Agent SDK

### Module 8: LangChain & LangGraph (RAG)
- LangChain introduction & basic concepts
- LangGraph introduction (what, how it works)
- Introduction to RAG (Retrieval-Augmented Generation)
- RAG in LangChain
- **Agentic RAG in LangGraph** (advanced – agents + RAG together)

### Module 9: Customer Support AI Agent
- Building an AI-powered customer support agent (complete tutorial)
